Татлин

Tatlin is a Gcode and STL viewer for 3D printing

See

[screenshots]

Download

You can download this project in either zip or tar formats:

You can also clone the project with Git by running: 

$ git clone git://github.com/dkobozev/tatlin

Install

Tatlin depends on the following packages:

  • Python 2.6+
  • PyGTK 2.24
  • PyOpenGL
  • Python bindings for GtkGLExt
  • NumPy

Run

To launch Tatlin from the command line:

$ python tatlin.py

You can also provide a filename to load:

$ python tatlin.py foobar.stl
